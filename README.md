# Bacillus mycoides colonization in tomato

Repository containing scripts and processed datasets used in the manuscript:

Systemic colonization by _Bacillus mycoides_ triggers transient metabolic reprogramming and oxylipin responses in tomato

## Repository contents

- Colonization analyses
- Metabolomics analyses
- Hormone profiling
- RT-qPCR analyses
- Statistical analyses
- Figure generation

## Experimental overview

Tomato plants were inoculated with Bacillus mycoides strain O2 and sampled over a 10-day time course.

## Requirements

R >= 4.5.1

Packages:
- tidyverse
- ggplot2
- readxl
- pheatmap
- dplyr

## Scripts

Scripts are organized according to manuscript figures.
